# Angular Jazar
## Simple service for getting posts from jazar.org 
###Installation
	bower install angular-jazar --save
###BASED ON http://v2.wp-api.org/

###Factories
- JazarPosts
- JazarEspaciosComunes
- JazarPages
- JazarEvents
- JazarMedia
